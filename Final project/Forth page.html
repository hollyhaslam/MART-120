My Semester of Coding

I created an interactive game on Alice. A website to learn coding in a fun way. In my jungle the girl is lost and she has to interact with the animals and objects to have things happen for her to get home.

This was our project with p5.js to draw a self portrait with shapes.

In this next assignment we took the same self portrait and made some of our objects moving. In this one the title is getting bigger and smaller, my name is moving diagonally across the screen, the lips change and the eyes move back and forth.

We created a simple game in p5.js this week made with shapes. These objects move around the screen randomly. Also, we learned to place an object when and where the mouse is clicked. My purple ellipse has been placed by the mouse. Finally, when the red character, which is moved by the keyboard goes through the exit - "You Win" is displayed.

This week not much changed with our game. Although I did change some colors. The main focus was changing our code to functions.

Finally we learned a bit about arrays. This allowed us to write code that could easily be changed to create a multiple amount of objects without having to copy and paste code for each one. With an array I created many moving shapes with just the change of one number.
